# Agentic AI Platform

Production‑ready agentic platform with:
- Approval‑gated infrastructure automation
- Terraform‑based AWS provisioning
- Dynatrace observability automation
- FinOps cost ingestion (AWS Cost Explorer)
- ServiceNow incident creation

## Key Principles
- Humans approve every destructive action
- No secrets in source control
- All integrations use official APIs

## Quick Start
1. Fill `deploy/k8s/secrets.yaml`
2. Deploy backend + UI
3. Request approval
4. Execute actions via API/UI
